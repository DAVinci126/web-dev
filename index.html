Explore the web platform

The web platform is a constantly evolving ecosystem of browser features and APIs for developers to make
websites that are easy and delightful to use. These pages are pathways for you to dive into each part of the web
platform on web.dev—at your pace.

HTML

HTML is the document layer of the web,
providing structure and semantics for pages.

CSS

CSS is the presentation layer of the web, and
allows you to make your web pages look
exactly like you imagine.

JavaScript

JavaScript adds rich interactivity to web
pages, so you can build websites that help
users to perform tasks and accomplish goals.
